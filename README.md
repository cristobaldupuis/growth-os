Growth OS
Growth OS is an interactive execution framework and experiment tracking application designed to replace fragmented marketing spreadsheets with a rigorous, system-driven operating engine.

Built for multi-brand portfolios and high-velocity growth teams, it enables growth leaders, product managers, and marketing architects to prioritize high-leverage initiatives, maintain absolute data hygiene, and continuously calibrate performance estimates against actual revenue outcomes.

🏛️ Core Architectural Features
1. Multi-Brand Portfolio Management
Cross-Tenant Architecture: Seamlessly filter the dashboard, pipeline, and learning library across multiple specific brands, retailers, or product lines within a single workspace.

Knowledge Cross-Pollination: Centralized Learning Library allows insights generated by one brand to be instantly searchable and replicable by other teams in the portfolio.

2. Macro-Level Performance Dashboard
Capital Risk & Calibration Engine: Tracks total revenue impacted by completed tests alongside active revenue currently at risk. Automatically computes a running accuracy percentage comparing initial revenue estimates against realized values, preventing structural overestimation.

Executive Update Generator: One-click clipboard generation that instantly synthesizes active pipeline metrics, velocity, and calibration accuracy into a formatted update for stakeholder communication.

Velocity Metrics & Distribution: Visualizes workflow velocity via micro-sparkline charts (started vs. closed tasks) and real-time categorical breakdowns across Paid Media, Conversion, Retention, and Merchandising.

3. Integrated AI Optimization Toolkit (BYOK Secure)
Engineered with a Bring-Your-Own-Key (BYOK) architecture. Users input their Anthropic API key via the Settings menu, which is securely stored exclusively in local browser memory.

Quick Capture & Hypothesis Expansion: Connects to Claude to transform rough, raw ideas into mathematically precise, single-sentence growth hypotheses formatted to your specific business model.

Predictive ICE Scoring Assist: Evaluates contextual data variables to pitch unbiased Impact and Certainty values (1–10) complete with structural rationales to strip gut-feeling bias from prioritization matrices.

Automated Learning Synthesis: Scans all completed tests across the portfolio to automatically generate cross-functional, cross-retailer strategic insights.

4. High-Fidelity Experiment Lifecycle Tracking
Custom Initiative Sequencing: Automatically generates unique tracking IDs (e.g., NH-001) and allows distinct initiatives to be structurally linked to map downstream impacts and multi-cell operational sequencing.

Rigid Control Parameters: Dedicated fields protecting operational discipline, including defined primary metrics, actual resource/spend cost tracking for true ROI calculation, and hard kill criteria.

🛠️ Technology Stack & System Design
Runtime & Interface: React 18 / Vite (Scaffolded for zero-latency, modular layout rendering).

Design Language: Custom high-contrast, minimalist serif interface optimizing structural scannability.

State Persistence: Engineered with a robust, environment-agnostic data storage layer natively targeting window.storage (Claude Artifact sandboxes), browser localStorage (StackBlitz/Production), or localized memory fallbacks to preserve application state dynamically.

🚀 Local Development Setup
To run this project locally on your machine, ensure you have Node.js installed, then execute the following commands in your terminal:

Bash
# 1. Clone the repository
git clone https://github.com/cristobaldupuis/growth-os.git

# 2. Navigate into the directory
cd growth-os

# 3. Install dependencies
npm install

# 4. Spin up the local development server
npm run dev
Once the terminal flags the build as ready, open http://localhost:5173/ in your browser to view your local instance. Click the ⚙ icon to configure your workspace and add your Anthropic API token.
